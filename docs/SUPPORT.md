## 🐛 Report a bug

> *Found something not working as expected?*

**Before submitting:**
1. Check the [FAQ](/docs/basics/faq.md) for common issues
2. Search existing [GitHub Issues](https://github.com/pipepub/pipepub/issues)
3. Update to the latest version

**When reporting, include:**
- Clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Environment (OS, PipePub version)
- Relevant logs (enable `LOG_LEVEL=debug`)

```bash
LOG_LEVEL=debug ./tools/pipepub.sh publish
```

**[Report a bug →](https://github.com/pipepub/pipepub/issues/new/choose)**

<a id="request-a-feature"></a>

## ✨ Request a feature

> *Have an idea to make PipePub better?*

**Before requesting:**
- Check if the feature already exists
- Search existing [Feature Requests](https://github.com/pipepub/pipepub/issues?q=is%3Aissue+label%3Aenhancement)
- Consider if the feature aligns with PipePub's scope

**Include in your request:**
- Problem statement (what this solves)
- Proposed solution
- Alternative approaches you've considered

**[Request a feature →](https://github.com/pipepub/pipepub/issues/new/choose)**
